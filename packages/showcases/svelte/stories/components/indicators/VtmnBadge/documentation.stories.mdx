# Badge

## Quick start

You need to add `@vtmn/svelte` on your `package.json`

```
npm install @vtmn/svelte
```

Now, you can use `VtmnBadge` on your svelte file with

```svelte
<script>
  import { VtmnBadge } from '@vtmn/svelte';
</script>
```

## Value

There are 3 different types of display

- When value is undefined or 0
- When value is between 1 and 99
- When value is upper than 99

<Canvas>
