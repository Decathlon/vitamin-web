# VtmnTag

## Quick start

You need to add `@vtmn/svelte` on your `package.json`

```
npm install @vtmn/svelte
```

Now, you can use `VtmnTag` on your svelte file with

```svelte
<script>
  import { VtmnTag } from '@vtmn/svelte';
</script>
```
