# Shadows

> Shadows help us create hierarchies and emphasis. They improve understanding & user interactions.

For more details, see our <a class="vtmn-link vtmn-link--icon-along" href="https://www.decathlon.design/726f8c765/p/5522fa-shadows/b/69ed0e" target="_blank">Shadows' guideline</a>.

## Using [CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)

If you don't know CSS Custom Properties, you can see basic usage documentation [here](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties#Basic_usage). All our CSS Custom Properties for colors are listed at "Swatches" column just below:

```
var(--vtmn-shadow_100);
var(--vtmn-shadow_200);
var(--vtmn-shadow_300);
var(--vtmn-shadow_400);
```

## Using [Tailwind CSS](https://tailwindcss.com) utility classes

You can use shadows with Tailwind CSS utility classes about [Box Shadow](https://tailwindcss.com/docs/box-shadow). Here is the list of shadows extented by Vitamin:

<Source
  language="json"
  code={JSON.stringify(tailwindPreset.theme.extend.boxShadow, null, 2)}
/>

So, here is an example with associated code:
